計畫說明

FOSDEM 2024 即將在 2024/02/03 - 04 在比利時布魯塞爾舉辦，FOSDEM 為歐洲最大的自由與開源軟體開發者會議，每年吸引來自全世界的開源領域的參與者前往朝聖！

開放文化基金會在 FOSDEM 2023、2024 都有申請到社群攤位（Open Culture Foundation + Taiwan open source community）將帶著臺灣的開源社群一起前往 FOSDEM 2024！

CC-BY / petertc

什麼是 FOSDEM？

FOSDEM 完整名稱為 Free and Open source Software Developers' European Meeting，自由及開源軟體開發者歐洲會議，主要關注在自由軟體及開源軟體開發者的年度盛會。舉辦地點在比利時的布魯塞爾自由大學，活動形式有講座、議程、攤位、市集餐車等。活動無須報名與費用，約超過 2500 人參與。

召募開源社群

只要你參與的社群是基金會社群專案夥伴、社群合作夥伴、與 Open Source 有關或已為 COSCUP 2023 社群夥伴，都可給我們一點物資、訊息前往 FOSDEM，預計會在攤位介紹或是活動現場的佈告欄曝光與張貼社群訊息。

CC-BY-SA / Joseba Martos

開源社群定義

簡單來說只要是一群人推廣或作與開源有關的專案都可以算在「開源社群」，也不侷限在寫程式領域而已！ 如有任何疑問，都歡迎直接來信到 hi@ocf.tw 詢問。

介紹內容

請協助完成表單，提供一些資訊讓我們在攤位上可以更精準的介紹你們的社群所關注的領域或是未來即將舉辦的活動。

表單主要填寫的內容有：

  1. 社群名稱（中英文）
  2. 一句話介紹你的社群
    一段文字介紹你的社群，可以為副標題、專注的領域。中英文可。
  3. Logo 檔案
  4. 給開放文化基金會夥伴須知項目
    中文或英文，主要是給我們在攤位上介紹時可以說更多的內容給予參與會眾瞭解。
  5. 特別要宣傳的事項
    可以從上一題中取出項目是我們要特別分享的。
  6. 國際串連
    可以參考目前已知的攤位列表與 Devroom 議程，有哪些是與你們參與的社群相關聯，基金會夥伴有空的話會特別去拜訪並協助宣傳。

填寫社群資訊表單 最後填寫時間為：2023/12/28

填寫建議

建議先將問題充分了解後、都已有準備好的內容複製貼上到相對應的欄位中。

物資規格

傳單

由於 FOSDEM 的特色，佈告欄或是現場比較常出現傳單類型來宣傳或推廣資訊，建議製作傳單讓我們帶過去。傳單製作格式請用 A4 版面製作、最多二頁的內容。但請輸出成 A5 尺寸、雙面內容、可為色紙傳單。建議紙張不要超過 80 磅，總數不要超過 300 份。 如果沒把握內容呈現方式，可以先轉成 PDF 寄給我們協助檢視後再實體輸出！ 或是不確定哪裡可以輸出成色紙傳單，也都歡迎來信詢問我們！

可以參考基金會在擺攤時製作的傳單文宣樣式：

  • DM_CSOs_Handbook.pdf
  • DM_OONI.pdf
  • DM_OpenChain.pdf
  • DM_OSCVPass.pdf

貼紙

貼紙通常很快就會被索取一空，如有想要清庫存也可以提供給我們帶去！可以郵寄貼紙物資到開放文化基金會。

物資郵寄地址

傳單與貼紙請一同寄到基金會，地址為：

  • (105057) 臺北市松山區八德路四段 636 之 1 號

常見問題

Q：最後檢查
  1. 是否填寫了「社群資訊表單」？（最後填寫時間為：2023/12/28）
  2. 是否盤點了貼紙或製作新款貼紙？
  3. 介紹社群或開源專案的傳單是否輸出為 A5 尺寸？
  4. 是否將傳單與貼紙裝箱準備寄送到開放文化基金會？（最後收件時間：2024/01/10）
Q：最後收件期間

2024/01/05 - 2024/01/10 為任何郵寄物資的最後收件期間，因為還要評估個人行李與基金會物資，越早寄給我們越好規劃承載空間。

Q：貼紙傳單以外的文宣品

如果有貼紙、傳單以外的文宣品也想要我們帶去 FOSDEM 展示或是發送，歡迎拍張照片後寄信到 hi@ocf.tw 讓我們評估看看行李箱還有沒空間！

Q：物資返還

我們會盡可能的在 FOSDEM 將貼紙、傳單有效率而不浪費的方式發送，如有剩餘物資我們帶回來後，在後續基金會擺攤場合將繼續使用。 除非是無法重製的展示品，請再額外告知我們取回！
